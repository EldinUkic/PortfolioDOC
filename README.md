### 🖥️ 01 – Windows Server Lab

📂 Ordner: `01-Windows_Server_Lab`

**Inhalt:**  
In diesem Projekt wird eine realitätsnahe Windows-Unternehmensumgebung simuliert.  
Die Umgebung besteht aus einem Domain Controller (DC) sowie zwei angebundenen Clients.
Dabei werden zentrale Dienste wie Active Directory, DNS und DHCP eingerichtet und getestet, um typische Abläufe in einem Unternehmensnetzwerk nachzustellen. Es wird fortlaufend weiter ausgebaut um mehr Erfahrung zu sammeln.
